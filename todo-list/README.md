To-Do List Application:

Project Description:

The To-Do List Application is a simple Python-based project that allows users to manage their daily tasks efficiently. Users can add tasks, view their task list, and organize their activities through a menu-driven interface. This project is designed for beginners to understand Python concepts such as lists, loops, functions, and user input handling.

 Features:

 Add new tasks to the list
 View all saved tasks
 User-friendly menu-driven interface
 Simple and lightweight console application
 Easy to understand and modify

 Technologies Used:

 Python 3
 Visual Studio Code (VS Code)
 Git & GitHub
